# ORIGO — Panel de Administración

Sistema de gestión para ventas en MercadoLibre y Tienda propia.

## Archivos

- `admin.html` — Panel de administración completo (requiere clave)
- `index.html` — Tienda pública visible para clientes

## Secciones del admin

- **Panel** — KPIs globales, alertas y top ventas
- **Catálogo** — Productos con importar/exportar CSV
- **Stock Actual** — Depósito propio + Full ML
- **Alertas** — Productos bajo mínimo
- **Movimientos** — Ingresos, bajas, envíos a Full
- **Compras** — Facturas/remitos de proveedores
- **Ventas** — Importador reporte ML + ventas tienda
- **Cuenta Corriente** — Deudas y pagos por proveedor
- **Análisis ML** — Rentabilidad por producto

## Uso

Abrí `admin.html` en el navegador. No requiere servidor.
Los datos se guardan en `localStorage` del navegador.

## Clave de acceso

Ver con el administrador del sistema.
